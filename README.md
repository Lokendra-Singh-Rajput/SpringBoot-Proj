# SpringBoot E-Commerce Application

This is a full-stack e-commerce application built using Spring Boot, React (Vite), and MySQL.

The project allows users to:
- View products
- Search products
- Add products to the cart
- Place orders
- Manage products (Add, Update, Delete)

## Tech Stack

Backend:
- Spring Boot
- Spring Data JPA
- MySQL

Frontend:
- React (Vite)
- Axios
- CSS

Tools:
- Git & GitHub
- VS Code
- Thunder-Client

## How to Run

### Clone the repository

```bash
git clone https://github.com/Lokendra-Singh-Rajput/SpringBoot-Proj.git
```

### Backend Setup

1. Open the `Search` folder.
2. Create `application.properties` using `application-example.properties`.
3. Update the MySQL username and password.
4. Run the Spring Boot application.

### Frontend Setup

```bash
cd t-ecom
npm install
npm run dev
```
